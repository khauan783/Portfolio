# Portfolio

Este projeto é um portfólio pessoal desenvolvido com HTML, CSS e JavaScript.

## Sobre o projeto
Este site reúne uma apresentação pessoal, uma galeria de artes, links para contato e uma identidade visual moderna e responsiva.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Como visualizar
Você pode abrir o arquivo `index.html` diretamente no navegador, ou acessar a versão online pelo GitHub Pages.
